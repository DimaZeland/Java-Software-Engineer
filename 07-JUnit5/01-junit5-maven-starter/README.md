# JUnit 5 Maven Starter

ℹ️ The project implements unit, integration and acceptance tests.

### ⚙️ The project uses:

- [Java SE 15](https://www.oracle.com/java/technologies/downloads/archive/)
- [Apache Maven](https://maven.apache.org/)
- [Lombok library](https://projectlombok.org/)
- [JUnit 5 framework ](https://junit.org/junit5/)
- [Mockito framework](https://site.mockito.org/)
- [AssertJ library](https://assertj.github.io/doc/)
- [Hamcrest library](https://hamcrest.org/)
